Power BI Setup Quick Start Guide

1. Open Microsoft Power BI Desktop.
2. Click Get Data → Text/CSV and choose mumbai_house_data_cleaned.csv.
3. Go to the View tab → Themes → Browse for Themes and select power_bi_theme.json.
4. Open power_bi_dax_measures.dax and copy the DAX measures into your Power BI table.
5. Refer to POWER_BI_DASHBOARD_GUIDE.md in the project directory for complete 3-page layout blueprints!
